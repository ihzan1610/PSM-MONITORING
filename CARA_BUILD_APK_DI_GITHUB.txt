CARA BUILD DEBUG APK DI GITHUB

1. Upload semua isi folder ini ke repository GitHub.
2. Pastikan file ini ada:
   .github/workflows/build-debug-apk.yml

3. Buka tab:
   Actions

4. Pilih:
   Build Debug APK

5. Klik:
   Run workflow

6. Setelah selesai, download artifact:
   PSM-Sawit-Pantau-debug-apk

7. Ekstrak artifact.
   File APK ada di:
   app-debug.apk

CATATAN:
- Ini debug APK untuk uji coba.
- Kalau HP menolak install, aktifkan "Install unknown apps".
- Aplikasi sudah bisa landscape jika rotasi HP aktif.
